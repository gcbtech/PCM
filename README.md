# PCM (PC Mover) - Personal File Migration Utility

PCM (PC Mover) is a standalone Windows GUI utility designed for computer refurbisher shops and end users to migrate personal files, personalization settings, and application preferences from an old computer to a new one.

PCM focuses strictly on migrating personal data, leaving complex application reinstallations aside to ensure a predictable, clean transfer.

---

## Two Supported Transfer Methods

*   ### Method 1: USB / External Drive Transfer
    *   **Predictable NTFS Formatting**: Automatically formats the selected external drive to NTFS to avoid 4GB filesystem limits (FAT32) and ensure clean writes.
    *   **Auto-Run Manifests**: Generates a JSON manifest next to the executable on the drive root. The executable automatically detects the manifest on the new computer to launch directly into Import Mode.
    *   **Self-Copy to Media**: Automatically copies itself to the root of the transport drive upon successful export, allowing you to plug and run with zero setup.

*   ### Method 2: Network Transfer (LAN / Direct Ethernet)
    *   **High-Throughput Socket Tuning (New in V1.5)**: Automatically scales TCP send and receive socket window buffers (SO_SNDBUF and SO_RCVBUF) up to 2MB. This optimizes bandwidth utilization over high-speed local networks or point-to-point ethernet links.
    *   **Inline Stream Compression (New in V1.5)**: Employs dynamic zlib inline compression for text-heavy folders (such as Documents, AppData configuration files, and browser settings), significantly reducing overall transmission times.
    *   **Ephemeral SSL/TLS Sockets**: Encrypts all in-transit files using robust, ephemeral 2048-bit AES-256 TLS certificates generated dynamically in memory.
    *   **UDP Discovery Beacon**: The receiving PC broadcasts a silent discovery beacon. The sending PC automatically discovers the target IP on the local subnet with zero IP configuration required.
    *   **Secure 6-Digit Pairing**: Authenticates the connection using a secure pairing code generated by the receiving PC to guarantee zero bad-actor interception.
    *   **Direct Cable Support**: Supports Auto MDIX point-to-point ethernet transfers. Simply plug a single LAN cable between both PCs, wait for Windows to allocate link-local IPs (~30s), and stream at full gigabit speeds.

## Advanced Configurations (New in V1.5)

*   ### Windows Settings & Personalization
    *   **Registry Theme Capturing**: Automatically exports HKCU theme registry paths, personalization settings, colors, cursors, and desktop backgrounds to silent registration files.
    *   **Wi-Fi Profiles Extraction**: Captures XML configurations containing SSID credentials from ProgramData to register them seamlessly on the new PC without password prompts.

*   ### Curated AppData Settings Scanner
    *   **Notepad++ Draft Preservation**: Captures unsaved active draft tabs and UI preferences so Notepad++ re-launches with all active sessions intact.
    *   **Microsoft Office & Outlook**: Migrates signatures, active templates, and PST email archives.
    *   **Web Browser Workspace Profiles**: Copies full profiles for Chrome, Edge, and Firefox, preserving active sessions, extensions, search history, autofill, and configurations.
    *   **VS Code Workspace & Extensions**: Preserves configs, keybindings, and extension directories.
    *   **Discord Storage**: Preserves local storage tokens so the user remains logged in on the destination PC.

*   ### Manual Custom File and Folder Selection (Introduced in V1.4)
    *   **Targeted Custom Selection**: Manually browse and select specific directories or individual files (such as local application folders inside Program Files or profile files not caught by default scans) for transfer.
    *   **Dependency Warning Disclaimers**: Automatically prompts caution warnings detailing Registry key, DLL dependency, or machine-locked setting risks, ensuring users explicitly confirm custom item migration.
    *   **Path Length Safeguards**: Intelligently flattens and maps deep subfolders on the backup drive under short, unique names to bypass Windows 260-character (MAX_PATH) limitations.
    *   **Elevated Original Restoration**: Reconstructs parent directory structures and restores custom folders/files to their exact original absolute locations (such as protected C:\Program Files system folders) on the target PC using elevated Admin privileges.

---

## Premium UX Features

*   **Custom Folder Checklist**: Choose exactly which categories (Desktop, Documents, etc.) you want to migrate with live sizing estimations.
*   **Multi-User Mapping**: Enumerates local Windows profiles and prompts for mapping selections when multiple accounts exist.
*   **Conflict Resolution Preferences**: Global preferences to Replace, Skip, or Keep Both (automatically append numbers, e.g. file (1).txt) duplicate files.
*   **Real-time Statistics**: Shows live transmission speed in MB/s, precise progress bars, dynamic ETA, and current file indicators.
*   **UAC Auto-Elevation**: Standard embedded manifest triggers the Windows UAC elevation popup immediately on start. This grants rights to format disks and programmatically insert/remove silent firewall rules without bothering the user.
*   **Rich Logging**: Produces clear migration logs on both the Desktop and the transport drive showing success, skips, and failures.

## Technical Architecture

```
PCM (PC Mover)/
├── main.py                  # Entry point & auto-mode routing
├── network_engine.py        # Secure TLS socket framing & UDP beacon layer
├── scanner.py               # Windows user account & folder scanning
├── settings_ops.py          # Personalization registry & Wi-Fi profile commands
├── appdata_scanner.py       # Discovery, sizing, and restoration mapping of app data
├── bookmarks.py             # Chrome, Edge, and Firefox bookmark extraction/injection
├── drive_ops.py             # Disk listing, NTFS quick formatting, and ejection
├── copy_engine.py           # Recursive file copy controller & logger
├── manifest.py              # Metadata JSON parser & validator
├── requirements.txt         # Project dependencies
├── build.py                 # PyInstaller standalone build pipeline
├── PCM.spec                 # Windows UAC admin-enabled compilation spec
└── gui/
    ├── app.py               # Main Tkinter master container & exit hooks
    ├── network_view.py      # Network Role selection, Pairing, Target & Progress screens
    ├── export_view.py       # USB scanning, checklist, formatting & progress screens
    ├── import_view.py       # USB detection, user mapping, and import progress screens
    └── components.py        # Curated aesthetic styles, widgets, and lists
```

## Installation & Running

### Requirements
*   **Operating System**: Windows 10 / 11
*   **Python**: 3.8+
*   **Dependencies**: `customtkinter`, `psutil`, `cryptography`

To run in development mode:
```bash
# Install dependencies
pip install -r requirements.txt

# Run application
python main.py
```

---

## Compiling standalone PCM.exe

PCM comes with an automated compilation pipeline to build a single standalone `.exe` that runs on any clean Windows machine without Python installed:

```bash
# Build standalone EXE
python build.py
```

Upon success, you can find the administrative, fully bundled executable at `dist/PCM.exe`.
